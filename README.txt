Hangman

Purpose:
   you will learn how to work with dictionaries, lists, and nested if statements. You will also learn how to work with the string and random Python modules. 
   
Quest:
   1. Make a program that will print words from file
   2. Use random module to pick one word and print
   3. Make user to play hangman( by quessing letters)
   4. Make a function to add new words
  
Dane:
   string
   random		https://docs.python.org/3/library/random.html
   dictionaries	https://docs.python.org/3/tutorial/datastructures.html
   lists		https://docs.python.org/3/tutorial/datastructures.html
   file operations	https://docs.python.org/3/tutorial/inputoutput.html
